# PlantUML C4 Diagrams

The diagrams in this folder are created using PlantUML and the C4 model for visualizing software architecture.

[https://github.com/plantuml-stdlib/C4-PlantUML](https://github.com/plantuml-stdlib/C4-PlantUML)

## Requirements

- Java
- PlantUML (installed as a Java .jar file)
- Graphviz (for rendering diagrams)
- An internet connection (for fetching the PlantUML C4 library)

## Usage

If you have the requirements installed, the diagrams will be generated automatically using a pre-commit hook. Make sure you have run `pre-commit install` in the root of the repository.
